Data Science

What is Data Science?

Statistics

  • Knowing how to apply mathematical models to data
  • Understand the data generating process (probability theory)
  • Construction of statistical estimators to model data

Simple Model

For a sample \(\{X_i, Y_i\}^n_{i=1}\), we claim that

\[ Y_i = \beta_0 + \beta_1 X_i + \epsilon_i \]

  • \(\beta_0\), \(\beta_1\): Regression Coefficients
  • \(\epsilon_i\): Error term

Goal

Construct

\[ \hat Y_i = \hat\beta_0 + \hat \beta_1 X_i \]
